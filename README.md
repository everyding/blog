# blog

## hexo构建博客系统

**Demo:** [everyding.github.io](https://everyding.github.io/)

该blog使用的主题是jane

## 使用方法

- 第一步.安装hexo所需插件
```cmd
npm install 
```

- 第二步.发布站点

```cmd
hexo deploy
```

- 第三步.启动服务本地预览

```cmd
hexo server
```

## 将blog部署到everyding.github.io命令

```cmd
hexo clean
hexo generate
hexo deploy
```
